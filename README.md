# Green Code Smell Package

A "green code smell" refers to code that works but may hide maintainability or scalability issues. This package demonstrates a simple example.

Installation:
```bash
pip install -e .
```

to run test (whole project): 
```bash
python src/green_code_smell/cli.py .
```

to run test (per folder/file): 
```bash
python src/green_code_smell/cli.py [folder or file name]
```
